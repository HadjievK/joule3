# SF Payroll AI Agent

An AI agent for SAP SuccessFactors payroll operations: natural language queries, anomaly detection, tax compliance validation, and compensation management

## Overview

Uses A2A Protocol, LangGraph, LiteLLM, and SAP Cloud SDK.

## Structure

- `app/main.py` - A2A server entry
- `app/agent_executor.py` - Request handling
- `app/agent.py` - Agent logic
